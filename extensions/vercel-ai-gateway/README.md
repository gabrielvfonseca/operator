# Vercel AI Gateway Operator provider

Official Operator provider plugin for Vercel AI Gateway.

## Install

```sh
operator plugins install @gabrielvfonseca/vercel-ai-gateway-provider
```

## Docs

See `docs/providers/vercel-ai-gateway.md` in the Operator repository, or the published docs at `https://docs.operator.ai/providers/vercel-ai-gateway`.
