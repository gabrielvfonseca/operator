# Operator Cohere Provider

Official Operator provider plugin for Cohere's OpenAI-compatible Compatibility
API.

Install from Operator:

```bash
operator plugins install @gabrielvfonseca/cohere-provider
operator gateway restart
```

Configure a Cohere API key, then select `cohere/command-a-plus-05-2026`,
`cohere/command-a-reasoning-08-2025`, `cohere/command-a-vision-07-2025`,
`cohere/command-a-03-2025`, or `cohere/north-mini-code-1-0`.
