# Roadmap

## Current Focus

- Security and safe defaults
- Bug fixes and stability
- Setup reliability and first-run UX

## Next Priorities

- Supporting all major model providers
- Improving support for major messaging channels
- Performance and test infrastructure
- Better computer-use and agent harness capabilities
- CLI and web frontend ergonomics
- Companion apps (macOS, iOS, Android, Windows, Linux)

## How to Influence the Roadmap

Open a [feature request](https://github.com/gabrielvfonseca/operator/issues/new?template=feature_request.yml)
or discuss ideas in Discord. High-value, maintainable proposals that align with
the project vision are prioritized.
