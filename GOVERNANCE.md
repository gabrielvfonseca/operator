# Governance

## Maintainer Team

Operator is maintained by the Operator Foundation team. Current maintainers
are listed in [`MAINTAINERS.md`](MAINTAINERS.md).

## Decision Making

- Routine bug fixes, docs, and small features: maintainer review + CI green
- Architecture or policy changes: maintainer consensus or lead maintainer sign-off
- Breaking changes: maintainer team review + changelog/update path

## Transparency

Public development happens on GitHub. Security issues follow the private
disclosure path described in [`SECURITY.md`](SECURITY.md).

## Becoming a Maintainer

See [`CONTRIBUTING.md`](CONTRIBUTING.md) for the maintainer application process.
