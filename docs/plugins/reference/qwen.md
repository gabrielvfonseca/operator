# Qwen plugin

Adds Qwen, Qwen Cloud, Model Studio, DashScope, Qwen Oauth, Qwen Portal, Qwen CLI, Qwen Token Plan, Bailian Token Plan model provider support to Operator.

## Distribution

- Package: `@gabrielvfonseca/qwen-provider`
- Install route: npm; ClawHub: `clawhub:@gabrielvfonseca/qwen-provider`

## Surface

providers: `qwen`, `qwencloud`, `modelstudio`, `dashscope`, `qwen-oauth`, `qwen-portal`, `qwen-cli`, `qwen-token-plan`, `bailian-token-plan`; contracts: `mediaUnderstandingProviders`, `videoGenerationProviders`
